--------------------------------------------------
IMPORTANT AUTHENTICATION RULE
--------------------------------------------------

VirJoy AI must use DUAL LOGIN VERIFICATION.

Users must login using BOTH:

1. Email account
2. Mobile number with OTP verification

Login flow:

Step 1:
User enters email

Step 2:
User enters mobile number

Step 3:
Send OTP to mobile number

Step 4:
Verify OTP

Step 5:
Account becomes active

IMPORTANT:
The app should NOT allow access using only email login.

Both email AND verified mobile number are required.

This is required to:
- reduce fake accounts
- reduce unlimited free credit abuse
- improve account security

--------------------------------------------------
ACCOUNT SYNC RULE
--------------------------------------------------

If the same user logs into:
- website
- Android app

using the same verified email + mobile number,
then:
- credits
- subscriptions
- generated videos
- history

must remain synced automatically.
